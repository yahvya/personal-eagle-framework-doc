Framework Sabo

Sabo est un framework PHP designé pour ressentir le fait d'écrire du PHP malgré l'utilisation d'un Framework.

Description rapide

  • Logique de rendu MVC

  • Vues (Blade, Twig ou classique)

  • Adapté aux petits et moyens projets
